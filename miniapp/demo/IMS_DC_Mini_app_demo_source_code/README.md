## Description
This project is a Mini-Application development example. Developers can refer to it to call operate IMS Data Channel and other capability APIs provided by the SDK.

## Compilation
- build the project:
   ```bash
   npm run build

## Packaging
Package the compiled files into a ZIP archive to create the Mini-Application. The index.html and properties.json files must be placed in the root directory of the ZIP package.
For reference, see:[miniapp/demo/IMS_DC_Mini_app_demo.zip](../IMS_DC_Mini_app_demo.zip)
